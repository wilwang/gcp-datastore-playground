# Simple Datastore bulk insert and bulk delete via batch 

## Setup virtual env
```
> virtualenv venv
> source venv/bin/activate
> pip install -r requirements.txt
```

## Modify main.py for insert or delete
```
#delete_all("TestObject")
main()
```

## Run code
```
> python main.py
```

## Deactivate venv
```
> deactivate
```
